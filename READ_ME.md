# Aurora

Aurora is a modern developer portfolio built with Vue 3,
TypeScript and Tailwind CSS.

Rather than listing technologies,
Aurora focuses on storytelling,
product thinking and thoughtful engineering.

## Tech Stack

- Vue 3
- TypeScript
- Tailwind CSS
- Vue Router
- Pinia

## Goals

- Fast
- Accessible
- Responsive
- Component-driven
- Production quality

## Development

```bash
npm install
npm run dev
```
